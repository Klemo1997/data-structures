# Data Structures Repository
This is my repo for training data structures core principles. All data structures are
implemented in C99 language.

Repository currently contains implementation of:
1. [Linked list](linked_list)
2. [Binary search tree](binary_search_tree)
3. [Trie](trie)
4. [Stack](stack)
5. [Queue](queue) (in progress)

## How to run it:

1. Change into desired DS dir: `cd [PARTICULAR_DIRECTORY]`
2. Run cmake: `cmake .`
3. Run make: `make`
4. Change into out dir: `cd out`
5. Run executable: `./example_executable`
6. Or run tests: `./test_example_executable`
